# Escompte Projet Finance

A web-based finance project focused on discount calculations and financial computations.

## Overview

This project is built with modern web technologies to provide an intuitive interface for financial operations, particularly focusing on discount (escompte) calculations.

## Tech Stack

- **JavaScript** (39.9%) - Core application logic and interactivity
- **HTML** (32.6%) - Page structure and markup
- **CSS** (27.5%) - Styling and layout

## Features

- Discount calculations
- Financial computations
- User-friendly web interface
- Responsive design

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Haythemidi02/Escompte_projet_finance.git
```

2. Navigate to the project directory:
```bash
cd Escompte_projet_finance
```

3. Open the project in your browser by opening the main HTML file.

## Usage

Open the application in your web browser and follow the on-screen instructions to perform financial calculations and discount operations.
